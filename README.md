# ABC Benchmark

ABC Benchmark, short for Attention Benchmark for Cognition, is a synthetic benchmark suite for attention evaluation across modalities.

The broader ABC design space spans:

- attention family
- attentional basis
- modality
- task type

The intended ABC benchmark scope covers:

- selective attention
- sustained attention
- attention shifting
- divided attention

This repository currently instantiates the selective-attention slice of that broader space with:

- attentional basis: feature-sensitive and structure-sensitive
- modality: text and visual
- task type: filtering and counting

The repo currently contains four dataset generators for selective attention:

- feature-sensitive text
- feature-sensitive visual
- structure-sensitive text
- structure-sensitive visual

Each generator produces task-ready examples for:

- counting: return how many items satisfy the rule
- filtering: return which items satisfy the rule

Visual datasets also render images, including ID-labeled variants for filtering tasks.

## Dataset Families

### Selective Attention Feature-Sensitive Text

Text records where the answer depends on matching explicit item features. Current slices include axes such as:

- `set_size`
- `rule_arity`
- `noise_width`
- `confound`
- `position`
- `target_count`
- `combined`
- additional adversarial and logical slices

Generated by:

- [scripts/generate_sa_feature_text_datasets.py](/Users/patka/code/PythonProject/abc-benchmark/scripts/generate_sa_feature_text_datasets.py)

### Selective Attention  Feature-Sensitive Visual

Rendered scenes where the answer depends on visual feature matching. Current slices include:

- `set_size`
- `rule_arity`
- `confound`
- `spatial_density`
- `target_count`
- `layout_regularity`
- `combined`

Generated by:

- [scripts/generate_sa_feature_visual_datasets.py](/Users/patka/code/PythonProject/abc-benchmark/scripts/generate_sa_feature_visual_datasets.py)

### Selective Attention Structure-Sensitive Text

Text layouts where the answer depends on structural grouping rather than flat feature matching. Current slices include:

- `principle`
- `target_count`
- `combined`

Principle variants currently cover:

- `paragraph_proximity`
- `section_common_region`
- `format_similarity`
- `scope_indentation`
- `continuation_chain`

Generated by:

- [scripts/generate_sa_structure_text_datasets.py](/Users/patka/code/PythonProject/abc-benchmark/scripts/generate_sa_structure_text_datasets.py)

### Selective Attention Structure-Sensitive Visual

Rendered scenes where the answer depends on visual grouping principles. Current slices include:

- `principle`
- `target_count`
- `combined`

Principle variants currently cover:

- `proximity`
- `similarity`
- `continuity`
- `common_region`

Generated by:

- [scripts/generate_sa_structure_visual_datasets.py](/Users/patka/code/PythonProject/abc-benchmark/scripts/generate_sa_structure_visual_datasets.py)

## Notes

- Dataset generation is deterministic per slice and seed.
- Visual generators write plain images and ID-labeled images for downstream evaluation.
- Structure-sensitive `target_count` slices are defined so the counting gold answer matches the number of matching items inside the anchor group.
